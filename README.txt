Set up the project.


DO THIS FIRST. Setup Android on your Eclipse.

http://developer.android.com/tools/sdk/eclipse-adt.html
http://developer.android.com/sdk/installing/installing-adt.html
http://developer.android.com/tools/help/adt.html


THEN DO THIS. ASAP method.

1. download a "libgdx-nightly-<date>.zip" from http://libgdx.badlogicgames.com/nightlies/

2. extract the zip file to a new folder called \flux-invaders\libgdx

3. in your eclipse ... file > new > Java Project.

4. in popup. Uncheck "Use default location".

5. browse to the \flux-invaders\gdx-invaders and open this project.

6. in your eclipse ... file > new > Project ...

7. in popup. Open the Android Folder and click "Android Project from Existing Code"

8. browse to \flux-invaders\gdx-invaders and open this project.

9. your eclipse should be free of red marks.

10. running ... haven't tried yet.
